About Project

It is an task management web-applicaton. With the help of this web application you can manage your task on daily basis.
And this application is created with the help of flask, Html and bootstrap.

It allows users to manage their tasks from a smartphone, tablet and computer. The technology is produced by the team behind Wunderlist
It's a list of tasks you need to complete, or things that you want to do. Most typically, they're organised in order of priority.
Traditionally, they're written on a piece of paper or post it notes and acts as a memory aid.
